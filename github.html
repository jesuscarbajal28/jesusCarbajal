GitHub es una plataforma de desarrollo colaborativo basada en Git que permite a los desarrolladores almacenar, gestionar y colaborar en proyectos de software, facilitando el control de versiones, la revisión de código, y la integración de flujos de trabajo automatizados mediante GitHub Actions. Con características como repositorios para el almacenamiento del código, herramientas para colaboración como pull requests y issues, y opciones para documentación y wikis, GitHub se ha convertido en un estándar para proyectos tanto de código abierto como privados. Fundada en 2008 por Tom Preston-Werner, Chris Wanstrath, P. J. Hyett y Scott Chacon, GitHub ha jugado un papel crucial en la evolución del desarrollo de software moderno. En 2018, Microsoft adquirió GitHub por 7,500 millones de dólares, aunque la plataforma sigue operando de manera independiente, sirviendo a millones de desarrolladores en todo el mundo.
